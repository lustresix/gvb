
# gvb

go实现的博客后端系统，提供了登录登出，文章发布，点赞评论等功能。使用gin作为http框架，使用gorm作为orm，使用redis，mysql，elasticsearch作为存储数据库。

## 在本地运行

Clone 这个 project

```bash
  git clone https://link-to-project
```

前往项目目录

```bash
  cd gvb
```

映射数据库并创建用户

```bash
  .\gvb db
```

启动服务器

```bash
  .\gvb
```

寻找其他配置项
```bash
  .\gvb -h
```
